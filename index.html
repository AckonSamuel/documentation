Introduction

Base URL: https://insure.imperialgeneral.com

This set of APIs provides a comprehensive range of functionalities for interacting with the core system for motor insurance policies:

  • Creating of customer
  • Get Vehicle Scheme: Determine whether a vehicle falls under the old or new rating scheme.
  • Compute Premiums for Old and New Rating Schemes: Calculate insurance premiums based on the old and new rating schemes.
  • Add Motor Policy: Create and add a new motor policy.
  • Search for Policy Details Using Phone Number: Retrieve policy information by searching for policy details associated with a specific phone number.
  • Get Vehicle Details for a Motor Policy: Retrieve vehicle details for a motor policy in the core system using either the phone number and vehicle registration number of the insurer or the reference number.
These APIs provide a robust and flexible framework for interacting with the core system, enabling seamless integration and efficient management of motor insurance-related operations.
As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

Api Keys You need to authenticate every API request you make to every Phrontlyne General Insurance API using Bearer Token. You can request for access token from Phrontlyne via email using the support@phrontlyne.xyz

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.
